About The Project
1. Import file jwdb.sql terlebih dahulu
2. Extract file JWDB-ProjectAkhir-NadiaSyachrani ke dalam C:>xampp:>htdocs
3. Pada halaman home terdapat 3 destinasi wisata di kota cirebon yaitu keraton kasepuhan, gua sunyaragi dan waterland ade irma suryani masing-masing terdapat harga
4. Jika pesan tiket di kli, akan menampilkan form untuk mengisi data
5. Klik hitung total bayar untuk menghitung total yang harus dibayar berdasarkan jumlah tiket, dengan harga tiket untuk setiap anak dihitung 50% dari harga normal
6. Jika sudah klik pesan tiket, maka akan menampilkan hasil dari pemesanan tiket, tombol home untuk kembali ke home, tombol data tiket untuk melihat hasil database pada data tiket
7. Pada halaman data tiket ini dapat dilakukan pemesanan tiket juga, fungsi nya sama, yang jika diklik pesan tiket akan menampilkan preview tiket nya
